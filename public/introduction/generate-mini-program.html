Giới Thiệu

Zalo Mini Program CLI là một công cụ giúp chúng ta có thể tạo và deploy nhanh một mini program

Nó cung cấp các chức năng như:

  • Tạo ứng dụng mới
  • Kiểm thử ứng dụng ở môi trường phát triển
  • Deploy ứng dụng lên hệ thống Zalo Mini Program

Cài Đặt

Đầu tiên bạn phải cài đặt Node.js (12.x) và NPM

Sau đó chúng ta có thể cài đặt Zalo Mini Program CLI từ NPM:

$ npm install -g zmp-cli

Sử Dụng

zmp <command> [options]

Tạo Ứng Dụng Mới

Tạo thư mục chứa project

mkdir mini-app && cd mini-app

Chạy lện tạo mini program

zmp create

Bạn sẽ được yêu cầu login với các thông tin sau:

  • Zalo App ID: App ID lấy được ở bước Đăng Ký Mini Program
  • Zalo Access Token: Truy cập Zalo for Developer và đăng nhập với tài khoản Zalo. Chọn Công cụ -> API Explorer -> Chọn ứng dụng tương ứng với App ID -> Lấy User Access Token -> Lấy mã

Sau khi login thành công bạn cần điền các thông tin để tiến hành tạo mini program

Kiểm thử ứng dụng ở môi trường phát triển

Chạy lệnh sau ở ứng dụng vừa tạo

zmp start

Deploy ứng dụng lên hệ thống Zalo Mini Program

Để có thể chạy ứng dụng trên Zalo chúng ta chạy lện sau

zmp deploy

Sau khi deploy thành công chúng ta dùng Zalo để quét mã QR truy cập ứng dụng.
